sbt-rats provides a plugin for the simple-build-tool that enables the
Rats! parser generator to be used in Scala projects.

For more information:

* sbt-rats: sbt-rats.googlecode.com
* Rats!: http://cs.nyu.edu/rgrimm/xtc/rats.html
